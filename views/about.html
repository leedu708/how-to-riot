How-To: Riot Games API

The focus of this guide is to utilize the User Stats API provided by Riot Games and to build a library of functions that can utilize the data from the API. The current documentation provided by Riot Games only shows developers which URLs are needed to send requests to the API. Along with the given API call, I will be writing a guide on how to access each inidividual stat for each player in every region and sum up the how-to with creating global statistics on each champion in the company's game, League of Legends.

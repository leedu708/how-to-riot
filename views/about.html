How-To: Riot Games API

The focus of this guide is to utilize the User Stats API provided by Riot Games and to build a library of functions that can utilize the data from the API. The current documentation provided by Riot Games only shows developers which URLs are needed to send requests to the API. Along with the given API call, I will be writing a guide on how to access each inidividual stat for each player in every region and sum up the how-to with creating global statistics on each champion in the company's game, League of Legends. Riot Games API

Content of the How-To, Guidelines

This how-to focuses on areas which are lacking in the official documentation provided by Riot Games. Given the API calls that I am working with, I will be writing several functions that leverages Riot's API. These functions include:

Summoner Recent Matches

This function will execute API calls to summoner-v1.4 and game-v1.3 in order to find a player's recent match history based on the player's summoner name.

Summoner Stats

This function will execute API calls to summoner-v1.4 and stats-v1.3 to find a player's ranked stats with each champion based on the player's summoner name.

Champion Settings

Thsi function will execute API calls to lol-static-data-v1.2 and champion-v1.2 to find the different settings each champion has set based on the champion's name.

Riot's API

The primary calls this project makes to Riot's API are:

  • summoner-v1.4
  • game-v1.3
  • stats-v1.3
  • lol-static-data-v1.2
  • champion-v1.2

summoner-v1.4

Returns information about a specific player given the player's summoner name or player ID.

game-v1.3

Returns information about a player's recent match history given the player's ID.

stats-v1.3

Returns information about a player's ranked statistics given the player's ID.

lol-static-data-v1.2

Returns all champions and their relevant data including the champion's name and ID.

champion-v1.2

Returns the settings of a champion as currently set up in the client given the champion's ID.
